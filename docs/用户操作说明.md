# 用户操作说明

- 标题栏双击：收缩/展开窗口至仅标题栏/恢复原尺寸。
- 收缩按钮：点击标题栏右侧“收缩”按钮，效果同双击。
- 快捷键：按下 `Alt+S` 可在收缩与展开间切换。
- 操作栏换行：当窗口变窄，顶部操作栏会自动换行以保证可读性；变宽时恢复为单行。
- 最小宽度：窗口最小宽度降低至约 160px（受控件固定宽影响，窄宽下将自动竖排）。
- 列宽调整：
  - 首列随窗口自适应（占据剩余空间）。
  - “优先级/日期/完成”列默认宽分别为 80/50/50，支持拖动表头调整。
- 测试运行：在项目根目录执行 `python -m unittest discover -s tests -p "test_*.py" -q`，可验证上述交互行为。

提示：所有新功能均与原有任务增删改查、排序、分页和动画兼容。
